# 匹兹堡睡眠质量指数量表（PSQI）AI辅助评估问答设计

📋 评估说明
开始前向患者说明：
"接下来我想了解一下您最近一个月的睡眠情况。请您根据这一个月大部分时间的真实情况来回答，不是指某一天的特殊情况。"

---
🗣️ 问答设计（共18个问题）
第一部分：睡眠时间相关
问题1：上床时间
- 提问："您最近一个月，通常是晚上几点上床睡觉的？"
- 回答方式：开放式，记录具体时间（如：晚上11点）
  问题2：入睡时间
- 提问："您上床后，一般需要多长时间才能睡着？"
- 选项：
    - A. 15分钟以内
    - B. 16-30分钟
    - C. 31-60分钟
    - D. 超过60分钟
      问题3：起床时间
- 提问："您通常是早上几点起床的？"
- 回答方式：开放式，记录具体时间（如：早上7点）
  问题4：实际睡眠时间
- 提问："您每天晚上实际睡了多长时间？（不是躺在床上的时间，而是真正睡着的时间）"
- 回答方式：开放式，记录小时数（如：6小时）

---
第二部分：睡眠困扰情况
前导语："接下来我问您一些睡眠中可能遇到的问题，请告诉我最近一个月这些情况发生的频率。"
频率选项说明：
- 0 = 没有（最近一个月没发生过）
- 1 = 偶尔（每周少于1次）
- 2 = 有时（每周1-2次）
- 3 = 经常（每周3次或以上）

---
问题5a：入睡困难
- 提问："您有没有躺下后30分钟内睡不着的情况？"
- 选项：0-3分（频率选项）
  问题5b：夜间醒来
- 提问："您有没有半夜醒来或者凌晨早醒的情况？"
- 选项：0-3分
  问题5c：夜尿
- 提问："您有没有因为要上厕所而起床的情况？"
- 选项：0-3分
  问题5d：呼吸不畅
- 提问："您睡觉时有没有呼吸不顺畅、憋气的感觉？"
- 选项：0-3分
  问题5e：咳嗽或打鼾
- 提问："您睡觉时有没有咳嗽或者打呼噜很响的情况？"
- 选项：0-3分
  问题5f：感觉寒冷
- 提问："您睡觉时有没有感觉太冷的情况？"
- 选项：0-3分
  问题5g：感觉燥热
- 提问："您睡觉时有没有感觉太热的情况？"
- 选项：0-3分
  问题5h：做噩梦
- 提问："您有没有做噩梦的情况？"
- 选项：0-3分
  问题5i：疼痛不适
- 提问："您有没有因为身体疼痛或不舒服而影响睡眠的情况？"
- 选项：0-3分
  问题5j：其他原因
- 提问："除了我刚才问的这些，还有没有其他原因影响您的睡眠？如果有，请告诉我是什么原因，多久发生一次？"
- 回答方式：开放式描述 + 频率选项（0-3分）

---
第三部分：睡眠药物使用
问题6：安眠药使用
- 提问："您最近一个月有没有吃过安眠药或其他帮助睡眠的药物？包括医生开的药和自己买的药。"
- 选项：
    - 0 = 没有吃过
    - 1 = 偶尔吃（每周少于1次）
    - 2 = 有时吃（每周1-2次）
    - 3 = 经常吃（每周3次或以上）

---
第四部分：日间功能
问题7：日间困倦
- 提问："您最近一个月，白天有没有因为睡眠不好而感到困倦、打瞌睡、没精神的情况？"
- 选项：
    - 0 = 没有
    - 1 = 偶尔有（每周少于1次）
    - 2 = 有时有（每周1-2次）
    - 3 = 经常有（每周3次或以上）
      问题8：做事精力
- 提问："您最近一个月，有没有因为睡眠不好而影响做事的精力和积极性？比如工作、学习或日常活动时提不起劲？"
- 选项：
    - 0 = 没有影响，精力很好
    - 1 = 有一点影响
    - 2 = 影响比较明显
    - 3 = 影响很大

---
第五部分：睡眠质量总体评价
问题9：睡眠质量自评
- 提问："总的来说，您觉得最近一个月自己的睡眠质量怎么样？"
- 选项：
    - 0 = 很好
    - 1 = 比较好
    - 2 = 比较差
    - 3 = 很差

---
第六部分：同床伴侣观察（如适用）
前导语："如果您有同床睡觉的家人或伴侣，我想了解一下他们观察到的情况。如果您是一个人睡的，这部分可以跳过。"
问题10a：打鼾
- 提问："您的家人/伴侣有没有说过您睡觉时打呼噜很响？"
- 选项：
    - 0 = 没有
    - 1 = 偶尔（每周少于1次）
    - 2 = 有时（每周1-2次）
    - 3 = 经常（每周3次或以上）
    - 不适用（一个人睡）
      问题10b：呼吸暂停
- 提问："您的家人/伴侣有没有发现您睡觉时呼吸会停顿、暂停的情况？"
- 选项：同上0-3分或不适用
  问题10c：腿部抽动
- 提问："您的家人/伴侣有没有发现您睡觉时腿会抽动、踢腿的情况？"
- 选项：同上0-3分或不适用
  问题10d：其他异常
- 提问："您的家人/伴侣有没有发现您睡觉时其他不正常的表现？比如说梦话、梦游等？"
- 回答方式：开放式描述 + 频率选项

---
📊 计分说明（供AI后台使用）
PSQI包含7个维度：
| 维度 | 计分方法 | 分值范围 |
| :--- | :--- | :--- |
| **C1：主观睡眠质量** | 问题9得分 | 0-3分 |
| **C2：入睡时间** | 问题2 + 问题5a综合 | 0-3分 |
| **C3：睡眠时间** | 根据问题4实际睡眠时长 | 0-3分 |
| **C4：睡眠效率** | (实际睡眠时间/在床时间)×100% | 0-3分 |
| **C5：睡眠障碍** | 问题5b-5j总分 | 0-3分 |
| **C6：催眠药物** | 问题6得分 | 0-3分 |
| **C7：日间功能障碍** | 问题7 + 问题8综合 | 0-3分 |
总分：7个维度得分相加，范围0-21分
- ≤7分：睡眠质量良好
- >7分：存在睡眠问题
