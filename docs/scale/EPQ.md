# 艾森克人格问卷（EPQ）AI辅助评估问答设计

📋 评估说明
开始前向患者说明：
"接下来我想了解一下您的性格特点。我会问您一些关于您平时习惯、想法和行为的问题。这些问题没有对错之分，请根据您真实的情况回答。如果这个描述符合您的情况，就回答'是'；如果不符合，就回答'否'。请不要想太久，凭第一感觉回答就好。"

---
📏 评分标准
所有问题使用二分法：
- 是 = 符合我的情况
- 否 = 不符合我的情况
  ⚠️ 重要提示：
- 不要犹豫太久，凭第一印象回答
- 没有"有时是，有时否"的选项，必须选择更接近的一个
- 如果实在难以选择，想想"大多数时候"是怎样的

---
🗣️ 问答设计（88题完整版）
版本说明
EPQ有多个版本：
- EPQ-88：成人版（88题）- 最常用
- EPQ-100：完整版（100题）
- EPQ-RSC：简版（48题）- 快速筛查
  以下以EPQ-88为标准设计

📊 计分方法
1. 原始分计算
   各维度原始分 = 该维度所有题目得分之和
   这是根据图片内容转换的 Markdown 格式表格。

| 维度 | 题号 | 满分 |
| :--- | :--- | :--- |
| **E（内外向）** | 1-21 | 21分 |
| **N（神经质）** | 22-45 | 24分 |
| **P（精神质）** | 46-65 | 20分 |
| **L（掩饰性）** | 66-88 | 23分 |

---
2. T分数转换
   原始分需要转换为T分数（标准分）才能解释：
   T分数公式：
   $$T = 50 + 10 \times \frac{X - \bar{X}}{SD} $$
   其中：
- X = 个人原始分
- X̄ = 常模平均分
- SD = 常模标准差
  中国常模参考值（成人）：

| 维度 | 男性均值±SD | 女性均值±SD |
| :--- | :--- | :--- |
| **E** | 11.48±4.76 | 12.18±4.70 |
| **N** | 11.04±5.56 | 13.86±5.68 |
| **P** | 6.14±3.54 | 4.66±3.06 |
| **L** | 13.62±4.74 | 14.88±4.56 |

---
3. T分数解释

| T分数范围 | 程度 | 百分位 |
| :--- | :--- | :--- |
| **≥70** | 非常高 | >98% |
| **60-69** | 偏高 | 84-98% |
| **40-59** | 中等（正常） | 16-84% |
| **30-39** | 偏低 | 2-16% |
| **<30** | 非常低 | <2% |
